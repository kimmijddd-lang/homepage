## 🎨 디자인 시스템

### 컬러 팔레트

```css
/* Primary Colors */
--bg-dark: #0d0d0f;           /* 메인 배경 (다크) */

/* Accent Colors */

/* Glass Effect */
--glass: rgba(255,255,255,0.05);

/* Text Colors */
--white: #ffffff;
--gray-300: #d1d5db;

--gray-500: #6b7280;
```

## 🧩 섹션별 상세 내용

### 1. 메인 (Hero)

**메인 카피**
```
세금 관리를 스마트하게
```

**서브 카피**
```
복잡한 세무를 데이터 기반으로 분석하고,
최적의 절세 전략을 설계합니다.
```

**비주얼 요소**
- 우측에 절세 효과 카드 (글래스모피즘 스타일)
- 그래프 애니메이션
- 배경: 그리드 패턴 + 블루/민트 글로우
